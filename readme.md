# CellNet
## Model
### Before using
You need to install the following environment:
```bash
torch 1.8.0+cu111
torchvision 0.9.0+cu111
opencv
pycocotools
```

### Train
```python
python train.py
```

### Evaluation
This work uses the following open source code to verify the performance of the model:
[mAP](https://github.com/Cartucho/mAP)
